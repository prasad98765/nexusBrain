## Color Palette

### Gradients
```css
/* Primary Features */
from-indigo-500 to-purple-600   /* AI Search, Upload */
from-purple-500 to-pink-600     /* Themes, Configure */
from-pink-500 to-red-600        /* Caching */
from-cyan-500 to-blue-600       /* Embed, Code */
from-pink-500 to-cyan-600       /* Final step */

/* Backgrounds */
bg-slate-800/30                 /* Section backgrounds */
bg-slate-800/50                 /* Card backgrounds */
bg-slate-900/50                 /* Nested elements */
bg-slate-900/80                 /* Preview cards */
```

### Text Colors
```css
text-white          /* Headings */
text-slate-100      /* Nav items */
text-slate-300      /* Body text (light) */
text-slate-400      /* Body text (muted) */
text-indigo-300     /* Accent text */
text-green-400      /* Success states */
```

## Responsive Breakpoints

```css
/* Mobile First */
< 768px   : 1 column layouts, stacked cards
768-1024px: 2 column grids, simplified layouts
> 1024px  : 4 column grids, full features

/* Key Features */
Mobile:  1 column (full width)
Tablet:  2 columns
Desktop: 4 columns

/* How It Works */
Mobile:  Vertical stack, no connection line
Tablet:  3 columns, arrows visible
Desktop: 3 columns, connection line + arrows

/* What Is Section */
Mobile:  Stack text above visual
Tablet:  Stack text above visual
Desktop: 2 columns side-by-side
```

## Animation Timings

```css
Hero Text Typing:    100ms per character

Hover Scale:        300ms transform
Border Transitions: 300ms all
Button Hover:       300ms all
Slider Transition:  4000ms auto-advance (disabled)
```
